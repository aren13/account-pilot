# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [Unreleased]

### Added

- Pydantic v2 configuration system with `password_cmd` resolution
- Data models and SQLite metadata database
- IMAP client with email parser
- Maildir sync engine
- IMAP IDLE daemon for real-time push
- Event system for hooks and automation
- Xapian full-text search indexer and query engine
- JWZ thread reconstruction
- Tag manager with auto-tag rules
- SMTP client and email composer
- MailPilot unified async API
- Click CLI interface
- Multi-account support (Gmail, Outlook, generic IMAP)
- Apache 2.0 open-source scaffolding
