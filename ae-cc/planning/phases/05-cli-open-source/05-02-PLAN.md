<task type="auto">
  <name>Task 2: Create GitHub templates and CI pipeline</name>
  <files>.github/workflows/ci.yml, .github/ISSUE_TEMPLATE/bug_report.md, .github/ISSUE_TEMPLATE/feature_request.md, .github/PULL_REQUEST_TEMPLATE.md, .gitignore</files>
  <action>
  CI pipeline (`.github/workflows/ci.yml`):
  ```yaml
  name: CI
  on: [push, pull_request]
  jobs:
    lint:
      - ruff check src/ tests/
      - ruff format --check src/ tests/
    typecheck:
      - mypy src/mailpilot/
    test:
      strategy:
        matrix:
          python-version: ["3.11", "3.12", "3.13"]
      steps:
        - Install system deps (xapian-core, python3-xapian)
        - pip install -e ".[dev]"
        - pytest --cov=mailpilot --cov-report=xml
    build:
      - pip install build
      - python -m build
      - Verify wheel and sdist created
  ```
  Use ubuntu-latest. Install xapian via `apt-get install -y python3-xapian xapian-tools libxapian-dev`.
  Add publish job: on tag push, publish to PyPI using trusted publishing (OIDC).

  Bug report template (`.github/ISSUE_TEMPLATE/bug_report.md`):
  - Labels: bug
  - Sections: Describe the bug, To Reproduce, Expected behavior, Environment (OS, Python, MailPilot version, IMAP provider)

  Feature request template (`.github/ISSUE_TEMPLATE/feature_request.md`):
  - Labels: enhancement
  - Sections: Is this related to a problem?, Describe the solution, Alternatives considered

  PR template (`.github/PULL_REQUEST_TEMPLATE.md`):
  - Sections: Description, Type of change (bug fix/feature/breaking), Checklist (tests, lint, docs)

  `.gitignore`:
  - Python standard: __pycache__, *.pyc, *.egg-info, dist/, build/, .eggs/
  - IDE: .vscode/, .idea/, *.swp
  - Testing: .coverage, htmlcov/, .pytest_cache/
  - MailPilot data: .mailpilot/ (never commit user data)
  - OS: .DS_Store, Thumbs.db
  - Env: .env, .venv/, venv/
  </action>
