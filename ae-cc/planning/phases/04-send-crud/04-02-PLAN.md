  `class MailPilot`:

  `__init__(self, config_path: Path | None = None)`:
  - Load config via `load_config(config_path)`
  - Initialize lazily — components created on first use (don't connect to IMAP just to do a local search)

  `async def initialize(self)`:
  - Create Database, initialize schema
  - Create SearchQuery (read-only Xapian)
  - Create SearchIndexer (for write ops)
  - Create EmailThreader
  - Create TagManager
  - Create EventEmitter
  - For each account: create ImapClient, SmtpClient (lazy — connect on first use)

  Context manager: `async with MailPilot() as mp: ...`

  Read operations:
  - `async def search(self, query: str, limit: int = 20, offset: int = 0, sort_by: str = "relevance") -> list[dict]`:
    1. Search Xapian for mp_ids + relevance
    2. For each mp_id: fetch from SQLite (full Message data)
    3. Get tags for each message
    4. Return list of SearchResult.to_json() dicts

  - `async def show(self, mp_id: str) -> dict`:
    1. Get message from db
    2. Read full body from Maildir
    3. Get tags
    4. Return full Message dict with body_plain, body_html, attachment details

  - `async def show_thread(self, thread_id: str) -> list[dict]`:
    1. Get all messages with this thread_id from db
    2. Order by date ascending
    3. Return list of message dicts

  - `async def list_unread(self, account: str | None = None, limit: int = 20) -> list[dict]`:
    1. Search `tag:unread` (optionally filtered by `account:{name}`)
    2. Return SearchResult dicts

  - `async def count(self, query: str) -> int`:
    1. Xapian count query

  - `async def count_unread(self) -> dict`:
    1. For each account: count `tag:unread AND account:{name}`
    2. Return dict: `{account_name: count, ..., "total": sum}`

  - `async def get_attachment(self, mp_id: str, part_id: int, save_to: str | None = None) -> str`:
    1. Get message from db, read raw from Maildir
    2. Parse MIME, extract attachment by part_id
    3. If save_to: write to file, return path
    4. If not: write to temp file, return path

  - `async def get_raw(self, mp_id: str) -> bytes`:
    1. Get message from db, read raw bytes from Maildir

  - `async def events(self, event_type: str | None = None, since: str | None = None, limit: int = 50) -> list[dict]`:
    1. Delegate to event_emitter.get_events()

<task type="auto">
  <name>Task 2: Implement MailPilot API — write and management operations</name>
  <files>src/mailpilot/__init__.py (extend)</files>
  <action>
  Add write and management operations to the MailPilot class:

  Write operations:
  - `async def send(self, account: str, to: list[str], subject: str, body: str, cc: list[str] | None = None, bcc: list[str] | None = None, html: str | None = None, attachments: list[str] | None = None) -> str`:
    1. Find account config by name
    2. Build SendRequest
    3. Delegate to EmailComposer.compose_and_send()
    4. Return Message-ID

  - `async def reply(self, mp_id: str, body: str, reply_all: bool = False) -> str`:
    1. Find account for this message
    2. Delegate to EmailComposer.reply()

  - `async def forward(self, mp_id: str, to: list[str], body: str | None = None) -> str`:
    1. Find account for this message
    2. Delegate to EmailComposer.forward()

  Management operations (each must update IMAP + SQLite + Xapian + emit event):
  - `async def mark_read(self, mp_ids: list[str])`:
    1. For each mp_id: get message, get account
    2. IMAP: set_flags \Seen
    3. SQLite: update flags, remove "unread" tag
    4. Xapian: update tags (remove XTAG:unread)
    5. Maildir: update flag in filename (add S)
    6. Emit email_read event

  - `async def mark_unread(self, mp_ids: list[str])`:
    1. IMAP: remove_flags \Seen
    2. SQLite: update flags, add "unread" tag
    3. Xapian: update tags
    4. Maildir: update flag
    5. Emit event

  - `async def flag(self, mp_ids: list[str])`:
    1. IMAP: set_flags \Flagged
    2. SQLite + Xapian + Maildir: update
    3. Add "flagged" tag
    4. Emit event

  - `async def unflag(self, mp_ids: list[str])`:
    - Inverse of flag

  - `async def move(self, mp_ids: list[str], to_folder: str)`:
    1. For each: IMAP move_messages
    2. SQLite: update folder field
    3. Xapian: update XFOLDER term
    4. Maildir: move file to new folder directory
    5. Emit email_moved event

  - `async def copy(self, mp_ids: list[str], to_folder: str)`:
    1. IMAP copy_messages
    2. Sync the new copies (they'll get new UIDs)

  - `async def delete(self, mp_ids: list[str], permanent: bool = False)`:
    1. If not permanent: move to trash folder (use provider alias)
    2. If permanent: IMAP delete + expunge
    3. SQLite: set is_deleted=1
    4. Xapian: remove document
    5. Emit email_deleted event

  Tag operations (delegate to TagManager):
  - `async def tag(self, action: str, tags: list[str], mp_ids: list[str] | None = None, query: str | None = None)`:
    1. If mp_ids: add/remove tags on specific messages
    2. If query: tag_by_query
    3. Must provide either mp_ids or query

  Helper:
  - `_get_account_for_message(self, mp_id: str) -> tuple[AccountConfig, ImapClient]` — look up message's account_id, return config + client
  </action>
