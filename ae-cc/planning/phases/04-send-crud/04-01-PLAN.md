  <action>
  Create `SmtpClient` class wrapping aiosmtplib:

  `__init__(self, account: AccountConfig)`

  `async def connect(self)`:
  - Create aiosmtplib.SMTP with host, port from config
  - If encryption == "tls": use `use_tls=True`
  - If encryption == "starttls": connect plain, then `await smtp.starttls()`
  - If encryption == "none": connect plain (warn in log)
  - Login with resolved password from config (`resolve_password(account.smtp.auth)`)

  `async def send(self, message: email.message.EmailMessage) -> str`:
  - `await smtp.send_message(message)`
  - Return Message-ID from the sent message
  - Handle errors: retry up to 3 times with 2s backoff on transient failures (5xx codes)
  - On auth failure: raise SmtpAuthError
  - On permanent failure (4xx): raise SmtpError with details

  `async def disconnect(self)`:
  - Close SMTP connection gracefully (QUIT)

  Context manager support for connect/disconnect.

  `smtp/__init__.py`: export SmtpClient, SmtpError, SmtpAuthError
  </action>

<task type="auto">
  <name>Task 2: Implement email composer</name>
  <files>src/mailpilot/smtp/composer.py</files>
  <action>
  Create `EmailComposer` class:

  `__init__(self, db: Database, smtp_client: SmtpClient, imap_client: ImapClient, event_emitter: EventEmitter, account: AccountConfig)`

  `async def compose_and_send(self, request: SendRequest) -> str`:
  1. Build `email.message.EmailMessage`:
     - Set From: `{account.display_name} <{account.email}>` (or just email if no display_name)
     - Set To, CC, BCC from request
     - Set Subject from request
     - Set Date to current UTC
     - Generate Message-ID: `<{uuid4}@{account.email.split('@')[1]}>`
     - Set body: plain text via `set_content()`, optional HTML via `add_alternative()`
     - Attach files: for each path in request.attachments, read file, `add_attachment()` with correct MIME type
  2. If request.in_reply_to: set In-Reply-To header
  3. If request.references: set References header (space-separated Message-IDs)
  4. Send via smtp_client.send()
  5. Save to IMAP Sent folder: `imap_client.append_message(sent_folder, message_bytes, ["\Seen"])`
     - Use account folder alias for "sent" to get correct folder name
  6. Emit `email_sent` event
  7. Return Message-ID

  `async def reply(self, mp_id: str, body: str, reply_all: bool = False) -> str`:
  1. Get original message from db
  2. Build SendRequest:
     - to: original from_address (if reply_all: also include to/cc minus self)
     - subject: "Re: {original_subject}" (don't double-prefix if already starts with "Re:")
     - in_reply_to: original message_id
     - references: original references_hdr + original message_id
     - body: user's body + quoted original (format: "\n\nOn {date}, {from} wrote:\n> {original_body_lines}")
  3. Call compose_and_send
  4. Set \Answered flag on original message via IMAP

  `async def forward(self, mp_id: str, to: list[str], body: str | None = None) -> str`:
  1. Get original message from db
  2. Get raw message from Maildir
  3. Build SendRequest:
     - to: specified recipients
     - subject: "Fwd: {original_subject}" (don't double-prefix)
     - body: optional user body + "---------- Forwarded message ----------\nFrom: {from}\nDate: {date}\nSubject: {subject}\nTo: {to}\n\n{original_body}"
     - If original has attachments: include them
  4. Call compose_and_send

  `async def save_draft(self, request: SendRequest) -> str`:
  1. Build EmailMessage (same as compose_and_send but don't send)
  2. Save to IMAP Drafts folder via append_message with \Draft flag
  3. Return Message-ID

  Body quoting helper:
  - `_quote_body(body: str) -> str`: prefix each line with "> "
  - Handle None bodies gracefully
  </action>
