  Create `ImapClient` class wrapping aioimaplib:

  Constructor: `__init__(self, account: AccountConfig, sync_config: SyncConfig)`
  - Store config, initialize connection pool dict `_connections: dict[str, IMAP4_SSL]`

  Connection management:
  - `async def connect(self, folder: str = "INBOX") -> IMAP4_SSL` — get or create connection for this folder. Use `aioimaplib.IMAP4_SSL` for TLS, `IMAP4` + STARTTLS for starttls. Resolve password via `resolve_password()` from config module. SELECT the folder after login.
  - `async def disconnect(self, folder: str | None = None)` — close specific folder connection or all
  - `async def reconnect(self, folder: str)` — disconnect + connect with exponential backoff (base_delay from config, max_delay from config, jitter)
  - `async def ensure_connected(self, folder: str)` — check connection alive, reconnect if needed

  IMAP operations (all take folder param, ensure_connected before each):
  - `async def list_folders(self) -> list[str]` — LIST command, parse folder names
  - `async def fetch_uids(self, folder: str, since_uid: int | None = None) -> list[int]` — SEARCH for UIDs, optionally only UIDs > since_uid
  - `async def fetch_message(self, folder: str, uid: int) -> bytes` — FETCH uid (RFC822), return raw bytes
  - `async def fetch_headers(self, folder: str, uid: int) -> bytes` — FETCH uid (RFC822.HEADER)
  - `async def fetch_flags(self, folder: str, uid: int) -> list[str]` — FETCH uid (FLAGS)
  - `async def set_flags(self, folder: str, uids: list[int], flags: list[str])` — STORE +FLAGS
  - `async def remove_flags(self, folder: str, uids: list[int], flags: list[str])` — STORE -FLAGS
  - `async def move_messages(self, folder: str, uids: list[int], to_folder: str)` — MOVE or COPY+DELETE
  - `async def copy_messages(self, folder: str, uids: list[int], to_folder: str)` — COPY
  - `async def delete_messages(self, folder: str, uids: list[int], permanent: bool = False)` — if permanent: STORE \Deleted + EXPUNGE, else move to trash
  - `async def append_message(self, folder: str, message: bytes, flags: list[str] | None = None)` — APPEND (for saving sent messages)

  Provider handling:
  - `providers/gmail.py`: `GmailProvider` class with folder alias resolution ("[Gmail]/Sent Mail", etc.), trash folder name, archive behavior
  - `providers/outlook.py`: `OutlookProvider` class with folder alias resolution ("Sent Items", "Deleted Items"), trash folder name
  - `providers/__init__.py`: `get_provider(name: str)` factory function
  - Provider is used by ImapClient to resolve folder aliases and handle provider-specific IMAP quirks (e.g., Gmail labels vs folders)

  Error handling:
  - Wrap all IMAP operations in try/except
  - On connection errors: log warning, attempt reconnect
  - On auth errors: log error, raise `AuthenticationError`
  - Custom exceptions in `src/mailpilot/imap/__init__.py`: `ImapError`, `AuthenticationError`, `ConnectionError`

<task type="auto">
  <name>Task 2: Implement RFC822 email parser</name>
  <files>src/mailpilot/imap/parser.py</files>
  <action>
  Create `EmailParser` class using mail-parser library:

  `parse_message(raw_bytes: bytes) -> dict`:
  - Parse raw RFC822 bytes using `mailparser.parse_from_bytes()`
  - Extract and return dict with all fields matching the Message model:
    - `message_id`: from Message-ID header
    - `from_address`: sender email
    - `from_name`: sender display name (or None)
    - `to_addresses`: JSON-encoded list of recipient emails
    - `cc_addresses`: JSON-encoded list (or None)
    - `bcc_addresses`: JSON-encoded list (or None)
    - `subject`: decoded subject (handle RFC2047 encoded words)
    - `date`: parsed datetime (use python-dateutil for RFC2822 dates — they're wild)
    - `in_reply_to`: In-Reply-To header value (or None)
    - `references_hdr`: References header as JSON list of Message-IDs
    - `preview`: first 200 chars of plain text body, stripped of whitespace
    - `has_attachments`: bool
    - `attachment_info`: JSON list of {filename, content_type, size} dicts
    - `size_bytes`: len(raw_bytes)

  `parse_body(raw_bytes: bytes) -> tuple[str | None, str | None]`:
  - Returns (plain_text_body, html_body)
  - Prefer plain text. If only HTML, don't convert — just return both.
  - Handle charset encoding gracefully — try declared charset, fallback to utf-8, fallback to latin-1

  `parse_references(header_value: str | None) -> list[str]`:
  - Parse the References header into a list of Message-IDs
  - Handle various formats: space-separated, newline-separated, with/without angle brackets
  - Return empty list if None

  Handle gracefully:
  - Malformed emails (parse what you can, don't crash)
  - Missing headers (return None for optional fields)
  - Encoding issues (multiple charset fallbacks)
  - Broken MIME structures (mail-parser handles most of this)
  </action>

  <done>Parser extracts all fields from raw RFC822 bytes, handles encoding edge cases, and never crashes on malformed input.</done>
